# Portfolio Homepage

A simple portfolio homepage built for [The Odin Project](https://www.theodinproject.com/).

## Overview

This project is a personal portfolio website designed to showcase projects, skills, and contact information. It was developed as part of The Odin Project curriculum. The website is also designed with a strong focus on accessibility to ensure a great experience for all users.

## Technologies Used

- **HTML5**: For semantic page structure.
- **CSS3 / Tailwind CSS**: For responsive and modern styling.
- **JavaScript (ES6+)**: For interactive elements.
- **Webpack**: For bundling modules and assets.
- **PostCSS**: For CSS transformations.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/StilettoDagger/homepage-project.git
   ```
2. Navigate to the project directory
   ```bash
   cd homepage-project
   ```
3. Install NPM packages
   ```bash
   npm install
   ```
4. Start the development server
   ```bash
   npm start
   ```

The application will automatically open in your default browser at `http://localhost:8080`.

## Scripts

- `npm start`: Starts the development server.
- `npm run build`: Builds the app for production to the `dist` folder.
- `npm run build:dev`: Builds the app for development.
- `npm run deploy`: Deploys the application to GitHub Pages.
- `npm run prettify`: Formats code using Prettier.
- `npm test`: Runs ESLint and Jest tests.
