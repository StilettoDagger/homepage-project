My Work

  • Battleship

    A fully playable Battleship game against a computer opponent. It utilizes Test Driven Development (TDD) with Jest, factory functions for core game logic, and an interactive DOM for the game boards.

  • Calculator

    An interactive calculator application with a clean UI. This project demonstrates core JavaScript logic, DOM manipulation, state management, and handling user inputs to perform accurate arithmetic operations.

  • Etch-A-Sketch

    A browser-based sketching tool inspired by the classic toy. It showcases dynamic DOM creation using JavaScript loops, event listeners for mouse hover effects, and allows users to customize the grid density on the fly.

  • Library

    A digital library for tracking reading progress. This application introduces Object-Oriented Programming principles in JavaScript using classes and objects, handling persistent data logic through HTML Web Storage API.

  • Restaurant Page

    A Single Page Application (SPA) utilizing modular JavaScript and Webpack. Content is dynamically generated using DOM manipulation, eliminating page reloads and providing a smooth, App-like user experience.

  • To-Do List

    A comprehensive task management app utilizing the module pattern and factory functions. It features separate logic, DOM, and data modules, supporting distinct projects, priority levels, and persistent local storage.
